매출기록시트 날짜 범위 표기 패치

적용 파일:
- stock-app-main/app.py

반영 내용:
1. 자동 매출기록에서 최신 날짜를 추가할 때, 기존 최신 기록일과 목표일 사이에 빈 날짜가 있으면 A열 날짜를 범위로 기록합니다.
   - 기존 최신 날짜 4/24, 목표 날짜 4/26 -> 4/25~4/26
   - 기존 최신 날짜 4/25, 목표 날짜 4/26 -> 4/26
   - 기존 최신 날짜 4/10, 목표 날짜 4/26 -> 4/11~4/26

2. 이미 4/25~4/26 같은 범위 날짜 행이 있고 같은 목표일을 다시 실행하면, A열의 범위 표기는 유지하고 B/C만 갱신합니다.

3. 기존 기능 유지:
   - 수동 매출계산 유지
   - 자동 매출계산 유지
   - 연도/날짜 Drive 폴더 기준 유지
   - 합계 아래로 들어가지 않도록 최신 날짜 행 위치 계산 유지
   - D:E 수식/서식 복사 유지
